# FRAMEWORK DE MIGRAÇÃO TASY

Este repositório segue uma metodologia estruturada para projetos de migração TASY (Delphi → HTML5).

## Estrutura Padrão de Projeto

Todo projeto deve conter:

01_Planejamento
02_Execucao
03_Governanca
04_Encerramento_e_Licoes_Aprendidas

## Fluxo Metodológico

1. Diagnóstico Inicial
2. Planejamento detalhado
3. Execução por módulos
4. Governança ativa (status, riscos, decisões)
5. Encerramento formal
6. Registro de lições aprendidas

## Princípios

- Foco em aderência de processo
- Controle de consumo de horas
- Gestão ativa de riscos
- Comunicação estruturada com cliente
- Formalização de decisões

## Objetivo

Criar previsibilidade, reduzir retrabalho e elevar maturidade em projetos de migração TASY.
